🏠 AI Smart Home Monitoring System

📍 Project Summary

This project is an AI-enabled Smart Home Monitoring System that delivers real-time motion detection, cloud-integrated alerting, and a web-based dashboard. I designed and implemented the full pipeline — from Python-based motion detection to Microsoft Power Automate flows and a SharePoint-integrated UI.
🔗 GitHub Repository

💡 My Role & Key Contributions

  • Built a Python application using OpenCV to detect motion events via webcam.
  • Designed and triggered Power Automate flows using HTTP POST to automate alerts and data logging.
  • Developed a SharePoint Framework (SPFx) Web Part using React and Fluent UI for the dashboard.
  • Configured SharePoint Lists and Document Libraries for storing logs and base64-encoded images.
  • Integrated ThingSpeak for simulating IoT events (optional testing layer).
  • Enabled real-time monitoring, notifications, and system scalability for future enhancements.
