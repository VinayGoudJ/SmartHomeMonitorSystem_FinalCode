AI Smart Home Monitoring System

by Vinay Kumar Goud Jagiri

Project Summary

This is a real-time, AI-powered Smart Home Monitoring System using Python, Power Automate, SharePoint, and SPFx. I designed the system from backend detection to frontend visualization and cloud automation.

GitHub Repository: SmartHomeMonitorSystem_FinalCode

System Architecture

The system follows a modular architecture with real-time motion detection on the edge and cloud-based alerting and visualization:

  • Device Layer: Python + OpenCV motion detector running locally
  • Data Transfer Layer: HTTP POST to Power Automate
  • Processing Layer: Power Automate flows parse JSON, trigger conditional alerts, and upload image evidence
  • Storage Layer: SharePoint Lists and Document Libraries
  • Presentation Layer: SPFx WebPart on SharePoint with charts, logs, and image viewers
  • Optional Integration: ThingSpeak IoT simulation for future AI/ML use cases

Application Development Lifecycle

This project followed an agile sprint-based model using a simplified SDLC approach:

  1. Requirement Gathering: Identified need for affordable AI home monitoring
  2. Design: Defined SharePoint schema, Power Automate logic, and SPFx UI structure
  3. Implementation: Developed modules independently: detector, flow, UI, and data store
  4. Testing: Motion detection stress-tested with variable lighting; email alerts tested by timestamp triggers
  5. Deployment: Hosted on SharePoint Online; flows published to cloud; GitHub project made public
  6. Review & Enhancement: Integrated feedback, improved UI layout, and planned ML pipeline integration

Key Contributions

  • Implemented Python OpenCV-based motion detection.
