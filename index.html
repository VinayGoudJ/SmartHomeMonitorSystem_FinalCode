AI Smart Home Monitoring System

by Vinay Kumar Goud Jagiri

Project Summary

This is a real-time, AI-powered Smart Home Monitoring System using Python, Power Automate, SharePoint, and SPFx. I designed the system from backend detection to frontend visualization and cloud automation.

GitHub Repository: SmartHomeMonitorSystem_FinalCode

Key Contributions

  • Implemented Python OpenCV-based motion detection.
  • Created Power Automate Flows for real-time alerting and SharePoint integration.
  • Built SPFx dashboard UI using Fluent UI, React, and CSS Grid.
  • Uploaded images to SharePoint Document Library via base64 encoding.
  • Simulated IoT inputs using ThingSpeak.
